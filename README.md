#  CSS 3D Magic Collection

Welcome to my CSS 3D playground! This repo showcases three stunning 3D projects built with pure CSS. Each project is a testament to what's possible with modern CSS3 transformations, animations, and a sprinkle of creativity.

### 1. Gran Turismo 3D Universe
Step into the world of Gran Turismo with this stunning 3D visualization. Inspired by the film, witness Jann Mardenborough standing alongside the Nissan chief who brought Gran Turismo to life. The scene comes alive with cars orbiting them like a high-speed solar system, showcasing the perfect blend of gaming and reality through smooth CSS 3D transforms.

### 2. Marvel 3D Showcase
Enter the Marvel Cinematic Universe with this dynamic 3D experience. At the center stands Thanos, the Mad Titan, wielding the power of the Infinity Stones. Watch as your favorite superheroes orbit around him in a cosmic dance, creating a breathtaking 3D spectacle that brings the epic battle for the universe to life through smooth CSS animations and transforms.

### 3. Interactive Soda Can 3D
Experience the magic of CSS with this interactive soda can that responds to your cursor. Hover over the can to trigger a smooth rotation while the mountain backdrop gracefully recedes, creating a stunning depth effect. This project perfectly demonstrates how CSS transforms and transitions can create engaging, interactive 3D experiences without a single line of JavaScript.
